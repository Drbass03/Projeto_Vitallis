# Projeto_Vitallis
Projeto pessoal desenvolvido com foco em prática e aprofundamento em banco de dados, especialmente em SQL Server.

O Vitallis simula um sistema de gestão para área da saúde, contemplando modelagem de dados, normalização, criação de objetos (tabelas, views, procedures, functions) e cenários reais de negócio.

Este repositório é evoluído continuamente conforme novos conhecimentos são adquiridos, servindo tanto como portfólio técnico quanto como material de estudo e consulta.
